## Getting Started

1. Clone the repo:

```bash
git clone https://github.com/n1cKar/invoice-app.git
cd invoice-app
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```
3. Run the development server:

```bash
npm run dev
# or
yarn dev
```
4. Open http://localhost:3000 in your browser to vist.
